# Peak Path

**Peak Path** is an interactive sports and wellness app built to help users track multi-sport activities, monitor fitness progress, encourage healthy behaviors, and enable child safety and focus features.

## Features

- **Multi-Sport Activity Logging:**  
  Search and select from a variety of sports (including yoga!), log sessions for each sport, and keep a persistent record of your activities.

- **Fitness Metrics and Progress:**  
  Tracks distance and calories burned across all logged activities. Provides warnings if users stop their sessions very early. Visualizes progress over time to encourage regular participation.

- **Behavior Rewards & Ranking:**  
  Awards points for positive habits such as phone lockdown mode, focused study sessions, and relaxation exercises. Includes a ranking system to motivate users.

- **Phone Lockdown & Relaxation Audio:**  
  Users can set a lockdown timer (options: 5, 10, 15, 20, 25, 30 minutes) to promote focus while also accessing breathing exercises and soothing background audio. Emergency call access is maintained as needed.

- **Child Location Tracking:**  
  Enables parents to track the location of their children for added safety and peace of mind.

- **Educational Resources Access:**  
  Quick links to resources like NCERT textbooks for easy educational support.

## Platform

- **Windows Application**  
  Designed primarily for Windows; integrates Python-based backend logic and a user-friendly interactive interface.

## Getting Started

1. **Download & Install:**  
   Follow the installer instructions provided in the repository or release page.

2. **Set Up Your Profile:**  
   Add basic information, select your favorite sports, and optionally set up parental tracking.

3. **Start Logging Activities:**
